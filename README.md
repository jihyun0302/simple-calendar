# simple-calendar
단순한 일정관리 앱 (개발 중)



## 할일

1. 달력 크기 비율에 맞게 변경

   - 비율이 달라질 경우 다른 형태의 달력으로 보여줌
   - 달력이 변경될 경우 투명 효과

2. 하루 일정 리스트 보여주기

   - 달력에 날짜를 클릭할 경우 일정리스트 열기

   - 시간에 따라 정렬하여 보여줌
   - 시각적으로 보여 줄 수 있도록

3. 일정 리스트 Paging

   - 페이지 처럼 다음 날짜로 넘길 수 있도록함
   - 만약 다음 달로 넘어 갈 경우 달력을 넘김

4. 일정 추가 프래그먼트 생성

   - 일정 추가

5. Tool Bar

   - 현재 보고 있는 년월 표시

   - Tool Bar로 일정 이동

6. SQLite연동
